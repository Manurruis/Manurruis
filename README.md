# Data Analyst | Data & Customer Experience | Business Operations

Trabajo en la intersección entre datos, experiencia de cliente y operaciones, en entornos donde el análisis tiene impacto directo en decisiones de negocio.
Colaboro con equipos y stakeholders para entender problemas, definir métricas clave y transformar datos en acciones concretas que mejoren productos, procesos y resultados.
Analizo datos reales para comprender el comportamiento de los usuarios, optimizar operaciones y apoyar decisiones estratégicas basadas en evidencia.
Este portafolio combina proyectos profesionales (anonimizados) y proyectos formativos, siempre con un enfoque práctico y orientado a negocio.

## Sobre este repositorio
Este portafolio tiene también un propósito educativo.
Aquí documento proyectos y ejemplos para:

- Socializar buenas prácticas en análisis de datos
- Mostrar casos prácticos reales
- Enseñar cómo enfocar proyectos hacia negocio

## Skills demostrables
Python • SQL • Tableau • Power BI • IA

## Proyectos Proyecto Júpiter — Benchmarking Mercadona

- Análisis competitivo del catálogo de supermercados en España
- Comparación de Mercadona con Carrefour, DIA, Lidl y Alcampo
- Integración de datos: API pública, datasets simulados y scraping de competidores
- Normalización de categorías, unidades, precios y marcas blancas
- Identificación de diferencias de precio, variedad y productos fuera de tendencia
- Recomendaciones para reforzar la competitividad de Mercadona

**Stack** : Python, Pandas, Web Scraping, Power BI

## EDA — Mercadona (Benchmarking de Supermercados)

- Análisis exploratorio del catálogo y precios de Mercadona
- Exploración inicial del catálogo obtenido vía API pública
- Identificación de patrones en precios, unidades, categorías y marcas blancas
- Detección de anomalías y productos fuera de tendencia frente al mercado
- Preparación del dataset para comparativas con Carrefour, DIA, Lidl y Alcampo

**Stack** : Python, Pandas, EDA
